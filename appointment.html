Appointment Manager

3.3.1. Appointment Manager
The Appointment Manager is a crucial system for outpatient management, representing the starting point for patient visits. It provides extensive capabilities for registration staff to view the availability of doctors, nurses, and equipment to efficiently distribute patients. The system supports managing multiple doctors simultaneously and offers the following features:

Key Features

Resource-Based Scheduling

Scheduling is based on available human and medical resources in the hospital.

Visual Calendar Interface

The system window features visual calendar and work schedule diagrams for intuitive use.

Order & Note Entry

Ability to enter orders and notes directly into the system.

Live Doctor Schedule Updates

Real-time updates to doctors' work schedules.

Efficient Search

Powerful search engines allow doctors to quickly access required data.

High Input Validation

High-level input validation and verification for all entries.

Visitor Types Managed

  • Walk-In Visitors: Patients arriving without a prior appointment, including emergency cases or inpatients needing specialist consultation.
  • By-Appointment Visitors: Patients with scheduled visits, either set directly or referred from other hospital departments.

Appointment Scheduling Functions

  • Identify patients by their unified ID provided with their card.
  • View daily appointment lists sent from hospital departments and confirm visit times.
  • Select a doctor or work within a group of doctors in the same specialty, with each patient called by the doctor.
  • View doctors' schedules at various levels: overview, monthly, weekly. The system suggests visit times for patients.
  • Doctor availability is based on both department and specialty schedules.
  • Send patients to the examination room with a single mouse click.
  • Instantly view the number of waiting patients.
